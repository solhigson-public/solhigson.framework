# Internationalization (.NET)

MUST use `IStringLocalizer<T>` for all user-facing text. MUST use `.resx` resource files per area (e.g., `Resources/Views/Events.resx`). MUST configure `RequestLocalizationMiddleware` to read culture from `Accept-Language` header and cookies. MUST use `CultureInfo.CurrentCulture` for date and number formatting. MUST NOT use culture-based currency formatting.

For setup, resource file structure, and backfill workflow, MUST invoke the `i18n` skill.
