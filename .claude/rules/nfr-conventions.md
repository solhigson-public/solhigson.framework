# Non-Functional Requirements Conventions

MUST use ID format `NFR-{CAT}-{NNN}`. Every NFR MUST have a measurable threshold and a verification method with tool and frequency. NFRs: `.claude/docs/nfr.md`, template: `.claude/templates/nfr-template.md`.

For structure rules, referencing conventions, and priority definitions, MUST invoke the `development-workflow` skill.
