# User Story Conventions

MUST use ID format `US-{AREA}-{NNN}`. MUST use "As a / I want / So that" format with 3-7 acceptance criteria in Given/When/Then. Stories: `.claude/docs/stories/{area}/`, template: `.claude/templates/user-story-template.md`.

For priority definitions, structure rules, and full conventions, MUST invoke the `development-workflow` skill.
